호텔 예약 관리 프로그램
메뉴별 시스템 구축 안내

1. 호텔 예약시스템을 구축한다.
호텔 예약시 입력한 호텔명, 룸, 체크인, 체크아웃 날짜를 기존 데이터베이스에 저장된 값과 비교하여
사용자가 원하는 날짜에 룸이 이미 예약이 되어있다면 중복 예약이 되지 않도록 한다.

2. 호텔별 예약 현황 확인 시스템을 구축한다.
원하는 호텔을 선택하면 해당 호텔의 예약 현황만 출력한다.

3. 개인 예약 정보 확인 시스템을 구축한다.
호텔 예약시 입력한 룸의 종류와 숙박일을 자동으로 계산하여 총 예약 비용을 사용자에게 출력한다.
(1박 기준 싱글룸 : 100,000원, 더블룸 : 300,000원, 트윈룸 : 400,000원, 스위트룸 : 650,000원)

4. 개인 예약을 취소하는 시스템을 구축한다.

5. 개인 예약 변경 시스템을 구축한다.
예약 시스템과 마찬가지로 사용자가 변경하고자 하는 날짜에 룸이 이미 예약이 되어있다면 변경이 되지 않도록 한다.
